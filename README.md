# ECEN4273-Project2
This repo covers the second project of ECEN 4273. This project is an artificial intelligence trained to recognize
 - Cats
 - Dogs
 - Drones
 - Pikachu
 - People
 
# Dependencies
 - sys
 - opencv
 - torch
 - torchvision
 - torchvision.io
 - matplotlib.pyplot
 - nummpy
 - torch.optim
 - os
 - pandas
 - skimage

# Current Performance
Currently, the AI is only able to label 3x100x100 images. It does not draw bounding boxes.
